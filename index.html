用hexo+githubpages搭建个人博客   2024-11-25

准备工作

  • GitHub 账户：确保你有一个 GitHub 账号。
  • Git 环境：本地安装好 Git，用于管理和推送代码。
  • 文本编辑器：推荐使用 VS Code，方便修改文件。

创建 GitHub Pages 仓库

  • 新建一个 GitHub 仓库：命名为 .github.io（将 替换为你的 GitHub 用户名）。
  • 启用 Pages 功能：默认情况下，这个仓库会自动作为你的 GitHub Pages 主机。

使用静态网站生成器

  • 选择一个静态网站生成器：Hexo 是一个流行的静态网站生成器，支持多种模板和插件。
  • 安装 Hexo：按照 Hexo 官方文档的指引，安装 Hexo 并初始化博客。

选择和使用模板

  • 选择一个模板：Hexo 提供了多个主题模板，你可以在 Hexo 主题列表 中选择一个。
  • 下载模板：将选定的模板下载到本地，并解压到 Hexo 博客目录的 themes 文件夹中。
  • 启用模板：在 Hexo 博客目录的 _config.yml 文件中，将 theme 设置为你下载的模板名称。
